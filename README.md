<p> This is my personal portfolio website built with <strong> <u> HTML, CSS, and JavaScript </u> </strong>. It is a simple and clean website that showcases my projects, skills, and work. </p>

Check out the [live website](https://daxshonly.github.io/My-portfolio/).

The project lives in the repository root and currently uses files such as `index.html`, `style.css`, `script.js`, and `Images/`.

## Development

Run the site locally from the project root:

```bash
git clone https://github.com/daxshonly/My-portfolio.git
cd My-portfolio
```

Then open `index.html` directly in your browser, or use a local development extension such as Live Server in VS Code. [page:2]

## Project Structure

```bash
My-portfolio/
├── index.html
├── style.css
├── script.js
├── medium-blog.js
└── Images/
```

## Customization

You can customize the portfolio by editing:

- `index.html` for content and sections
- `style.css` for styling and layout
- `script.js` for interactivity
- `medium-blog.js` for my blogs 
- `Images/` for images, icons, and screenshots.

## Deployment

This project can be deployed using GitHub Pages, Netlify, or Vercel, and the current README already points to a GitHub Pages live demo.
